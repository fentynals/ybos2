Privacy Policy

YUNGSBRUH.OS — the HEADTAP development dashboard. Last updated 5 July 2026.

1. Who this covers

YUNGSBRUH.OS ("the App") is a private, single-operator dashboard. It is used only by its operator to view analytics for the operator's own connected social accounts. It has no other users and does not collect data from the public.

2. What data we access

When the operator connects an account, the App reads, via the official platform APIs, only public and account-level statistics for that operator's own accounts, such as:

  • Follower / subscriber counts
  • Video and post view, like, comment, and share counts
  • Post titles and links

The App does not access private messages, contacts, or the data of any other user, and it does not post, modify, or delete content on the operator's behalf through the analytics sync.

3. How data is used and stored

Retrieved statistics are written into a data file (feed.json) that powers the operator's own dashboard display. Access tokens are stored as encrypted secrets in the operator's GitHub repository and are used solely to make read-only API calls. Data is not sold, shared, or transferred to any third party.

4. Third-party platforms

The App uses the TikTok, YouTube (Google), and Instagram (Meta) APIs. Data obtained from each platform is used in accordance with that platform's developer terms and policies.

5. Data retention & removal

Statistics persist only in the dashboard's data file and can be cleared at any time by the operator. To revoke access, the operator removes the stored tokens and disconnects the App from the platform's app settings.
